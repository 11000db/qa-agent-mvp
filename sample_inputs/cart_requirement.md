# 장바구니 기능 요구사항

## 기능 설명
사용자는 상품을 장바구니에 담고, 수량을 변경하거나 삭제할 수 있어야 한다.

## 사용자 스토리
- 사용자는 원하는 상품을 장바구니에 담아 주문 전까지 관리하고 싶다.
- 사용자는 동일 상품을 여러 개 담거나 필요 없는 상품을 제거하고 싶다.

## 상세 요구사항
1. 사용자는 상품 상세 페이지에서 장바구니 담기 버튼을 클릭할 수 있어야 한다.
2. 장바구니 담기 버튼 클릭 시 해당 상품이 장바구니에 추가되어야 한다.
3. 동일 상품을 다시 담으면 새로운 항목이 생기지 않고 수량이 증가해야 한다.
4. 사용자는 장바구니 페이지에서 상품 수량을 변경할 수 있어야 한다.
5. 상품 수량을 0으로 변경하거나 삭제 버튼을 누르면 장바구니에서 제거되어야 한다.
6. 장바구니에는 현재 담긴 상품 목록과 수량이 정확히 표시되어야 한다.
7. 장바구니 변경 사항은 사용자 액션 직후 화면에 반영되어야 한다.

## 예외 및 고려사항
- 품절 상품은 장바구니 추가 시 제한이 필요할 수 있다.
- 수량 변경 시 음수 또는 비정상 값은 허용되면 안 된다.
- 장바구니가 비어 있으면 빈 상태 메시지를 보여줘야 한다.

## 기대 결과 예시
- 성공 케이스: 상품 추가, 수량 증가, 수량 수정, 삭제 정상 반영
- 실패/예외 케이스: 비정상 수량 입력 방지, 빈 장바구니 상태 처리
