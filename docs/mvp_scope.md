# MVP Scope

## 포함 범위
이 프로젝트의 1차 MVP에서는 아래 기능만 구현한다.

- 요구사항 또는 기능 설명 텍스트 입력
- 입력된 요구사항에서 테스트 포인트 추출
- 리스크 기반 테스트 케이스 생성
- 우선순위(priority) / 리스크(risk) 분류
- 테스트 케이스를 구조화된 형태(JSON 등)로 저장
- 일부 테스트 케이스를 Playwright 테스트 초안으로 변환
- 샘플 시나리오 1~2개 실행
- 실행 결과(pass/fail, 로그, trace 등) 저장

## 제외 범위
이번 MVP에서는 아래 항목은 구현 대상에서 제외한다.

- 완전 자동 self-healing
- 사내 시스템 직접 연동
- CI/CD 파이프라인 완전 통합
- 멀티 에이전트 고도화
- 범용 엔터프라이즈 플랫폼 수준 UI
- 대규모 벡터DB / RAG 고도화
- 복잡한 권한 체계 및 다중 프로젝트 운영 기능

## MVP 성공 기준
아래 조건을 만족하면 MVP가 완성된 것으로 본다.

1. 샘플 요구사항을 입력할 수 있다.
2. 요구사항으로부터 테스트 포인트를 추출할 수 있다.
3. 테스트 케이스와 priority / risk를 생성할 수 있다.
4. 최소 1개 이상의 Playwright 테스트 초안을 만들 수 있다.
5. 최소 1개 이상의 샘플 테스트를 실행할 수 있다.
6. 실행 결과를 저장하거나 확인할 수 있다.
7. README와 데모 흐름이 정리되어 있다.
