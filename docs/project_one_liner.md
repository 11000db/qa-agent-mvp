# Project One-Liner

## 한 줄 소개
요구사항 문서를 입력받아 QA 관점의 구조화된 테스트 케이스 프롬프트와 JSON 산출물을 생성하는 AI 기반 QA Agent MVP

## 짧은 설명
이 프로젝트는 로그인, 회원가입, 장바구니 같은 이커머스 요구사항을 입력받아 테스트 포인트를 구조화하고, 재사용 가능한 프롬프트와 테스트 케이스 JSON 형태의 QA 자산으로 변환하는 것을 목표로 한다.

## 대외 한 줄 설명
수기 테스트 케이스 작성 경험을 AI 프롬프트·스키마·자동화 가능 구조로 바꾼 QA Agent MVP입니다.
